📌 프로그램 제목
SmartTime – 자바 기반 시간표 관리 프로그램

📌 프로그램 설명 (3줄 요약)
이 프로그램은 대학생들이 효율적으로 시간표를 구성하고 관리할 수 있도록 도와주는 콘솔 기반 애플리케이션입니다.
수업을 추가하거나 삭제하고, 시간 중복을 자동으로 감지하여 오류 없이 시간표를 구성할 수 있습니다.
또한 시간표를 저장하고 불러올 수 있어 반복적인 관리가 가능합니다.

📌 사용자 정의
대상 사용자: 대학교에 재학 중인 학생

사용 목적: 수업 시간표를 구성하고 효율적으로 관리하기 위해

📌 필요성
대학교 생활에서는 시간표가 매우 중요하며, 수업 시간 충돌이나 일정 관리를 정확히 하지 않으면 큰 불편이 생길 수 있습니다.
기존 종이 시간표나 단순 메모 방식은 시간이 지나면 불편하거나 잊기 쉬우며, 자동화된 관리가 어렵습니다.
따라서 간편하게 시간표를 추가, 확인, 저장할 수 있는 프로그램이 필요합니다.

📌 사용자가 이 프로그램을 사용해야 하는 이유는?
수업 간 시간 겹침 여부를 자동으로 검토하여 오류 방지

직관적인 콘솔 UI로 누구나 쉽게 사용 가능

저장 기능으로 매번 다시 입력하지 않아도 됨 (재사용성 보장)

📌 개발 방법
개발 언어: Java (JDK 11 이상 권장)

입출력 방식: 콘솔 기반

파일 저장: Object 직렬화를 사용하여 .txt 파일에 저장

핵심 기술: 클래스 설계(OOP), Java Collection(List), LocalTime API, 예외 처리

📌 전체 동작 과정
프로그램 시작 시 기존 시간표 파일을 자동으로 불러옴

사용자 메뉴 출력 (수업 추가, 삭제, 보기, 저장, 종료)

수업 추가 시 요일과 시간 겹침 여부 자동 체크

시간표 보기 기능으로 전체 일정 확인 가능

저장 기능으로 프로그램 종료 후에도 정보 유지

📌 동작 과정 설명
수업을 추가하면 내부적으로 리스트에 저장되며, 동일 요일·시간대의 수업과 겹치는지 검사

삭제 시에는 과목명으로 검색하여 삭제

시간표는 요일+시간 기준으로 정렬된 상태로 출력됨

저장 시 직렬화(ObjectOutputStream) 방식으로 데이터를 파일에 저장하며, 프로그램 시작 시 자동으로 복원

📌 개발 프로그램의 이점
실제 학교 생활에서 바로 사용할 수 있는 실용성

Java 언어의 클래스, 컬렉션, 파일 입출력 등 핵심 개념을 종합적으로 사용

콘솔 기반이라 가볍고 설치 없이 사용 가능

📌 사용자는 누구인가?
대상: 대학생, 특히 시간표를 자주 변경하거나 확인할 일이 많은 학생

적합한 사용자: 1~4학년 전체, 수강신청 후 시간표 구성이나 개인 학습 일정을 관리하고 싶은 사람

📌 그 사용자가 얻을 수 있는 이점은?
번거로운 시간표 작성과 관리에서 해방

수업 중복이나 실수 없이 깔끔하게 시간표 구성 가능

저장된 시간표를 언제든지 열람 및 수정 가능

프로그래밍 과제나 자기 프로젝트로 활용 가능 (코드 재사용도 가능
